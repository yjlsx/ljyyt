# 视频标签页布局说明

## 📐 当前布局结构

视频标签页的卡片布局已经是**封面图在上面，标题在下方**的设计：

```
┌─────────────────────────────┐
│                             │
│        [封面图片]           │  ← 封面图在顶部
│        (200px高度)          │
│                             │
│      [播放按钮图标]         │  ← 居中显示
│                             │
│      [时长显示]             │  ← 底部渐变背景
│                             │
├─────────────────────────────┤
│                             │
│      [视频标题]             │  ← 标题在封面下方
│                             │
│      [艺术家名称]           │  ← 艺术家信息
│                             │
└─────────────────────────────┘
```

## 🎨 HTML结构分析

```html
<div class="card video-card">
  <div class="card-body p-0">
    <!-- 封面图片区域 -->
    <div class="position-relative">
      <img src="封面URL" class="card-img-top" style="height: 200px; object-fit: cover;">
      
      <!-- 播放按钮（居中） -->
      <div class="position-absolute top-50 start-50 translate-middle">
        <i class="fas fa-play-circle text-white" style="font-size: 3rem;"></i>
      </div>
      
      <!-- 时长显示（底部） -->
      <div class="position-absolute bottom-0 start-0 end-0 p-2" 
           style="background: linear-gradient(to top, rgba(0,0,0,0.7), transparent);">
        <span class="text-white small">5:00</span>
      </div>
    </div>
    
    <!-- 标题和信息区域（在封面下方） -->
    <div class="p-3">
      <h6 class="card-title mb-1">视频标题</h6>
      <p class="card-text text-muted mb-0 small">艺术家名称</p>
    </div>
  </div>
</div>
```

## 📱 布局特点

### ✅ 封面图区域
- **位置**：卡片顶部
- **高度**：200px
- **样式**：`object-fit: cover` 确保图片填充
- **效果**：大封面设计，视觉冲击力强

### ✅ 播放按钮
- **位置**：封面图片中央
- **样式**：白色播放圆圈图标
- **大小**：3rem
- **透明度**：0.8

### ✅ 时长显示
- **位置**：封面图片底部
- **背景**：渐变黑色（从下到上透明）
- **文字**：白色小字

### ✅ 标题区域
- **位置**：封面图片下方
- **内边距**：p-3 (12px)
- **标题**：h6 标签，加粗
- **艺术家**：灰色小字

## 🎯 设计优势

1. **视觉层次清晰**
   - 封面图作为主要视觉元素
   - 标题信息作为次要信息

2. **空间利用合理**
   - 大封面展示视频内容
   - 下方区域提供详细信息

3. **用户体验良好**
   - 播放按钮位置明显
   - 时长信息不遮挡封面
   - 标题信息清晰可读

4. **响应式设计**
   - 在不同屏幕尺寸下保持良好布局
   - 移动端和桌面端都有良好体验
